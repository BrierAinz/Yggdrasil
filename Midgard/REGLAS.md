# 🌍 Reglas de Midgard - Productos Personales

> **Propósito:** Aplicaciones y herramientas para uso personal

---

## ✅ Sí Permitido
- Apps que uso yo (Piano, Portafolio, etc.)
- Scripts de automatización personal
- Herramientas de productividad
- Proyectos creativos personales

## ❌ Prohibido
- Código para terceros/clientes
- Proyectos comerciales
- Cualquier cosa con SLA/deadlines externos

## Reglas Específicas

1. **Funciona para mí**: No necesita ser perfecto, debe ser útil
2. **Sin presión externa**: No hay clientes ni deadlines
3. **Documentación mínima**: Lo justo para recordar cómo funciona
4. **Backup importante**: Son proyectos personales valiosos
