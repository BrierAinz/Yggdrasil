# 🏛️ Reglas de Asgard - Centro de Mando

> **Propósito:** Dashboards, monitoreo y control del ecosistema

---

## ✅ Sí Permitido
- Scripts de monitoreo (Python, bash, PS)
- Dashboards web (HTML/JS)
- Configuración central
- APIs de control
- Logs agregados

## ❌ Prohibido
- Lógica de negocio de proyectos
- Código de aplicaciones
- Datos sensibles sin encriptar
- Modificaciones directas a otros reinos

---

## Estructura
```
Asgard/
├── README.md
├── Dashboards/           # Paneles de control
├── Scripts/             # Scripts de gestión
├── Config/              # Configuración central
└── Logs/                # Logs agregados (no sensibles)
```

## Reglas Específicas

1. **Read-only preferente**: Observar, no modificar
2. **Conexión segura**: Usar APIs, no acceso directo a archivos
3. **Lilith se conecta aquí**: Asgard es su ventana al ecosistema
