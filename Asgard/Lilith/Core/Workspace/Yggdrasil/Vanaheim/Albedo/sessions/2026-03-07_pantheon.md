# Sesión: 2026-03-07 — Panteón de Agentes Lilith v2.1

## Misión
Implementar arquitectura multi-agente "Panteón" con:
- **LILITH** (Orquestadora) — grok-4-fast-reasoning — Ya existente
- **EVA** (Kimi) — Análisis, documentación, contexto largo
- **ADÁN** (Qwen local) — Código puro, sin internet
- **LUCIFER** (Venice) — Creativo, privado, peligroso

Regla absoluta: Lilith manda. Los demás ejecutan sin cuestionar.

## Agentes

### EVA — Analista (Kimi)
- **Modelo:** moonshot-v1-128k
- **API:** https://api.moonshot.cn/v1
- **Trigger:** contexto > 50k tokens, análisis, documentación
- **Keywords:** analiza, documenta, resume, audita, revisa todos
- **Personalidad:** Meticulosa, estructurada, sumisa. Nunca opina sin evidencia.
- **Formato:** Hallazgo → Evidencia → Recomendación

### ADÁN — Ejecutor (Qwen Local)
- **Modelo:** qwen2.5-coder:7b
- **API:** http://localhost:11434/api (Ollama)
- **Trigger:** código puro, tests, refactorización
- **Keywords:** genera código, escribe función, refactoriza, tests
- **Personalidad:** Pragmático, silencioso. Solo código limpio.
- **Reglas:** Sin preamble, sin explicaciones innecesarias.

### LUCIFER — Creativo (Venice)
- **Modelo:** llama-3.3-70b
- **API:** https://api.venice.ai/api/v1
- **Trigger:** creativo, privado, alternativas
- **Keywords:** creativo, imagina, inventa, privado, confidencial
- **Personalidad:** Rebelde, piensa fuera de límites. El más peligroso.
- **Formato:** Enfoque → Riesgos → Ejecución

## Routing Automático

El `AgentRouter` selecciona automáticamente el agente basado en la tarea:

```python
def select_agent(task: str, context_tokens: int = 0) -> str:
    # Eva: contexto largo o análisis
    if context_tokens > 50000 or "analiza" in task:
        return "eva"

    # Adán: código puro
    if "genera código" in task or "escribe función" in task:
        return "adan"

    # Lucifer: creativo/privado
    if "creativo" in task or "privado" in task:
        return "lucifer"

    # Default: Lilith/Grok
    return "grok"
```

## Uso

### Ejemplo 1: Análisis de código
```python
from Backend.core.agent_router import AgentRouter

router = AgentRouter()
result = await router.execute(
    task="Analiza todos los archivos del backend y dame un resumen",
    context=""  # Código fuente aquí
)
# → Delegado a Eva (Kimi 128k)
```

### Ejemplo 2: Generación de código
```python
result = await router.execute(
    task="Genera una función para cifrar sesiones"
)
# → Delegado a Adán (Qwen local)
```

### Ejemplo 3: Solución creativa
```python
result = await router.execute(
```

### Ejemplo 4: Explicar routing
```python
from Backend.core.agent_loop import AgentLoop

loop = AgentLoop()

```

**Error**: HTTP 401 - Invalid Authentication

**Keys probadas**:
- `sk-kimi-Qr83...` (original) ❌
- `sk-kimi-4TKr...` (nueva) ❌

**Modelos probados**:

**Diagnóstico**: La cuenta de Moonshot AI puede estar:
- Sin créditos/saldo
- Suspendida
- Requiere verificación adicional

**Fix requerido**: Verificar estado de cuenta en https://platform.moonshot.cn/ o crear cuenta nueva.
