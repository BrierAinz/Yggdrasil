# FASE 3: Swarm Coordination
## Plan de Implementacion

**Fecha:** 2026-05-01
**Dependencias:** FASE 1 (Skills), FASE 2 (Memory)
**Objetivo:** Multi-agent coordination para tareas paralelas

---

## Componentes

### 1. SwarmManager (`Lilith/Swarm/manager.py`)
- Spawn agents con task + capabilities
- Message bus para comunicacion
- File locking para evitar conflictos
- ConflictResolver para detectar/merge

### 2. SwarmAgent (`Lilith/Swarm/agent.py`)
- Worker con estado (IDLE, WORKING, REVIEWING, COMPLETE)
- File tracking (read/write sets)
- Code shift notifications
- Task execution

### 3. MessageBus (`Lilith/Swarm/message_bus.py`)
- Cola de mensajes entre agentes
- Tipos: task_complete, code_shift, code_shift_notice, lock_request

### 4. ConflictResolver (`Lilith/Swarm/conflict_resolver.py`)
- Detectar archivos modificados por multiples agentes
- Auto-merge cuando sea seguro
- Notificar cuando requiera intervencion

### 5. Tools (`Lilith/tools/swarm.py`)
- `spawn_swarm`: Crear swarm para una tarea
- `swarm_status`: Ver estado de agentes
- `swarm_result`: Obtener resultados

### 6. Tests (`Lilith/Swarm/tests/`)
- 15+ tests: spawn, message bus, locking, conflicts

---

## Comandos CLI (main.py)
- `/swarm spawn <task>` - Crear swarm
- `/swarm status` - Ver agentes activos
- `/swarm kill <id>` - Matar agente
