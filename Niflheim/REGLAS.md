# 🌫️ Reglas de Niflheim - Banco de Recursos

> **Propósito:** Assets, datasets, templates, dotfiles, configuraciones

---

## ✅ Sí Permitido
- Imágenes, iconos, fuentes
- Datasets para ML
- Templates de proyectos
- Configuraciones de herramientas
- Dotfiles (.bashrc, .gitconfig, etc.)

## ❌ Prohibido
- Código ejecutable (scripts de app van a sus proyectos)
- Archivos temporales
- Duplicados
- Archivos sin organizar

---

## Estructura
```
Niflheim/
├── README.md
├── Assets/             # Recursos visuales
│   ├── Images/
│   ├── Icons/
│   └── Fonts/
├── Datasets/           # Datos para ML
├── Templates/          # Plantillas reutilizables
│   ├── project_templates/
│   └── document_templates/
├── Configs/            # Configs de herramientas
│   ├── vscode/
│   ├── cursor/
│   └── terminal/
└── Dotfiles/           # Configuraciones de shell
```

---

## Reglas Específicas

1. **Recursos compartidos**: Usable desde cualquier reino
2. **Organizado por tipo**: Estructura clara
3. **Sin código ejecutable**: Solo datos/configs/assets
4. **Limpieza trimestral**: Eliminar assets no usados
