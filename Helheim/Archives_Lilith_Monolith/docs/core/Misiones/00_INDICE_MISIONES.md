## Cómo Agregar una Nueva Misión

1. Crear archivo `MISION_v{X.Y.Z}_{DESCRIPCION}.md`
2. Seguir el template de REGLAS_DOCUMENTACION.md
3. Actualizar este índice
4. Actualizar `00_INDICE_DOCUMENTACION.md` en la raíz de Docs
