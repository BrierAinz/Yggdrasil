# Aplicaciones de Midgard

Midgard es el reino que conecta Yggdrasil con el mundo humano. Aquí viven las aplicaciones finales.

## Aplicaciones Activas

### TerminalDashboard
Dashboard de terminal construido con Textual. Muestra el estado de los servicios, métricas de memoria y actividad de agentes.

### YggdrasilStudio
Interfaz web para interactuar con los agentes de Yggdrasil. Chat, gestión de memoria, visualización de herramientas.

### YggdrasilForge
Herramienta de generación de assets para el ecosistema. Integración con ComfyUI y Blender.

### Bots
- **Telegram Bot** — Interfaz conversacional via Telegram
- **Discord Bot** — Integración con servidores Discord

## Próximas Aplicaciones

- **Yggdrasil CLI v2** — CLI unificada con subcomandos por reino
- **Mobile App** — App móvil para monitoreo remoto
