# ᚹ Vanaheim — El Bosque de los Agentes

Vanaheim es el reino del framework de agentes. Aquí viven las abstracciones de alto nivel.

## Componentes

### vanaheim-framework
Framework para construir agentes compuestos. Proporciona:
- Interfaz unificada de agente
- Composición de capabilities
- Gestión de ciclo de vida
- Inyección de dependencias

### bifrost
Puente entre reinos. Permite la comunicación entre agentes de diferentes dominios.

## Filosofía

Los agentes en Vanaheim no son monolitos. Son composiciones de capabilities especializadas, cada una viviendo en su propio reino.
