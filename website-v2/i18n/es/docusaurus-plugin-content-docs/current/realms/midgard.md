# ᛏ Midgard — El Mundo de los Humanos

Midgard es el reino que conecta Yggdrasil con el mundo exterior. Aquí viven las aplicaciones finales.

## Aplicaciones

- **TerminalDashboard** — Dashboard de terminal con Textual
- **YggdrasilStudio** — Interfaz web de agentes
- **YggdrasilForge** — Generación de assets
- **Bots** — Telegram, Discord

## Filosofía

Midgard es la capa de presentación. No contiene lógica de negocio — solo traduce entre el ecosistema interno y el usuario humano.
