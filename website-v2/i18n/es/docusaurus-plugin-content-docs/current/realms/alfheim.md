# ᛇ Alfheim — La Tierra de la Luz

Alfheim es el reino de las interfaces de usuario y herramientas visuales.

## Componentes

### TerminalDashboard
Dashboard de terminal construido con Textual. Muestra en tiempo real:
- Estado de servicios
- Métricas de memoria
- Actividad de agentes
- Logs del sistema

### YggdrasilStudio
Interfaz web para interactuar con los agentes. Chat en tiempo real, gestión de memoria, visualización de herramientas.

### YggdrasilForge
Herramienta de generación de assets. Integración con ComfyUI para generación de imágenes y Blender para modelos 3D.
