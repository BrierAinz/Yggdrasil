# ᛚ Helheim — El Mundo de los Muertos

Helheim es el reino de los archivos. Aquí descansa el código legacy, las versiones anteriores y los experimentos abandonados.

## Contenido

- Versiones anteriores de Lilith (v1-v4)
- Experimentos abandonados
- Código que fue reemplazado por los paquetes modulares

## Principio

Lo que muere en Helheim no se borra — se archiva. El historial es conocimiento.
