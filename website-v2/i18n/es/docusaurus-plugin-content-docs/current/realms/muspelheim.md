# ᛗ Muspelheim — El Reino del Fuego

Muschelheim es el dominio de la creación y la generación. Aquí nacen los datos, los modelos y las experiencias.

## Módulos

### Horror GameMaster
Motor de juego de terror procedural con IA. Genera narrativas, NPCs inteligentes y gestión de tensión.
- 8 módulos: gamemaster, llm_engine, npc_intelligence, tension_manager, etc.
- ~2,200+ entradas JSONL de entrenamiento
- 19 modelos soportados para generación

### AI-Influencer
Pipeline de generación de contenido para redes sociales.

### ForgeMaster
Motor de generación procedural de contenido.

### AutoSub
Sistema de generación automática de subtítulos.

## Principio

El fuego crea, pero sin control destruye. Cada generación es auditada y validada.
