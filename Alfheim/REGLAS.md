# 🎨 Reglas de Alfheim - Laboratorio Visual

> **Propósito:** Prototipos de UI, experimentos frontend, diseño visual

---

## ✅ Sí Permitido
- Prototipos de UI (Figma exports, HTML/CSS/JS)
- Experimentos de diseño
- Themes y paletas de color
- Componentes React/Vue de prueba
- Animaciones y efectos visuales

## ❌ Prohibido
- Código de producción
- APIs backend
- Bases de datos
- Cualquier cosa que deba "funcionar" 24/7

---


## Reglas Específicas

1. **Todo es temporal**: Si dura >1 mes, migrar a Midgard/Asgard
2. **Numerar prototipos**: 001, 002, etc.
3. **README por prototipo**: Qué se probó, qué se aprendió
4. **Sin presión**: Fallar está permitido (y esperado)
