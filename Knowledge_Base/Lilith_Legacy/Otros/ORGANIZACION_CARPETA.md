# Organización de `Core/Docs` (Lilith 4.0)

Este archivo “ordena” la carpeta **sin mover archivos**: define categorías, convenciones y un mapa rápido para encontrar cada documento.

> Si más adelante quieres moverlos a subcarpetas (p. ej. `Core/Docs/01_Arquitectura/`), este mapa ya actúa como plan de migración.

---

## Convenciones recomendadas (sin renombrar hoy)

- **Misión**: `MISION_LILITH_*.md` (histórico por versión).
- **Deep dive**: `DEEP_DIVE_*.md` (investigación/decisiones detalladas).
- **Diseño**: `DISEÑO_*.md` (diseño implementable de un subsistema).
- **Roadmap/Plan**: `ROADMAP_*.md`, `PLAN_*.md`.
- **UX**: docs que describen experiencia Discord/operación.
- **Fixes**: `FIXES_*.md` o `fix_*.md` para issues concretos.
