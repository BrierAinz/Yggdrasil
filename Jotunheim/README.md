# Jotunheim — Reino de los Gigantes

**Proposito:** Proyectos masivos que exceden el alcance de un solo reino. Monorepos grandes, sistemas distribuidos, o integraciones complejas.

## Estructura
```
Jotunheim/
  projects/      — Proyectos masivos en desarrollo
  archives/      — Proyectos gigantes completados
```

## Reglas
- Solo proyectos que necesitan su propia infraestructura completa.
- Deben tener su propio README, CI/CD, y documentacion interna.
- No acumular proyectos pequenos aqui — usar reinos especificos.
