# Plataformas — Guía por Canal

## Instagram (@eir.creates)

### Setup Inicial
1. **Cuenta Profesional** — Obligatorio. Categoría: "Digital Creator"
2. **Bio**: "🌙 Dark fantasy AI artist | ✦ Forging images with runic code | 🗡️ Svartalfheim → Midgard"
3. **Link en bio**: Linktree con Patreon, CivitAI, Twitter
4. **Highlights**: Arte | Proceso | Estética | Sobre mí

### Formato de Posts
| Formato | Dimensión | Uso |
|---------|-----------|-----|
| Feed Portrait | 1080x1350 (4:5) | Retratos, estética |
| Feed Square | 1080x1080 (1:1) | Arte digital, close-ups |
| Reel | 1080x1920 (9:16) | Tutoriales, transiciones |
| Story | 1080x1920 (9:16) | BTS, polls, WIP |

### Algoritmo IG (lo que funciona en 2025)
- **Reels > Carousels > Single posts** (en orden de alcance)
- Postear entre 18:00-21:00 CET (horario de mayor actividad)
- Carousel de 5-10 imágenes: 3x más engagement que post simple
- **No usar más de 5 hashtags** (IG penaliza el spam de hashtags)
- Respuestas en comentarios los primeros 30 min = boost al algoritmo

### Anti-Ban Checklist
- [x] Cuenta profesional marcada como "Digital Creation"
- [x] Bio menciona explícitamente "AI artist"
- [x] No usar fotos de personas reales como referencia
- [x] Marcar cada post con #aigenerated o disclaimer en caption
- [x] No comprar seguidores ni engagement
- [x] Mismo dispositivo para login siempre (no VPN)

## TikTok (@eir.creates)

### Estrategia
TikTok es el **motor de crecimiento**. Un solo video viral puede dar 10K-100K seguidores.

### Tipos de Video
1. **Transición**: Foto borrosa → foto nítida con beat drop
2. **Proceso**: Screen recording de SD generando + resultado final
3. **Tutorial**: "Cómo hacer [efecto] en Stable Diffusion" (30-60 seg)
4. **Estética mood**: Montaje de 5-8 imágenes con música dark ambient
5. **Before/After**: Prompt básico → resultado con LoRA/controlnet

### Edición
- **CapCut** (gratis, suficiente para todo)
- Duración óptima: 15-30 segundos
- Hook en los primeros 2 segundos
- Texto overlay explicativo
- Audio trending + dark remix

### Posting
- 2-3 videos/semana mínimo
- Horario: 20:00-22:00 CET
- Crosspost a IG Reels automáticamente

## Twitter/X (@eir_creates)

### Estrategia
Twitter/X es para **comunidad AI art** — engagement técnico, no estético.

### Tipos de Contenido
1. **Thread educativo**: "🧵 Cómo logré este efecto [hilo de 5-8 tweets]"
2. **Prompt sharing**: Prompt + resultado + configuración técnica
3. **WIP**: Progreso de generación, iteraciones
4. **Comunidad**: Reply a otros AI artists, RT contenido de nicho
5. **Encuestas**: "¿Qué estilo prefieren? Dark fantasy vs Cyberpunk"

### Hashtags Específicos
```
#StableDiffusion #AIart #Midjourney #AIartist #GenerativeArt
#ControlNet #LoRA #DarkFantasy #ConceptArt #DigitalArt
```

### Frecuencia
- 3-5 tweets/día (mix de original + engagement)
- 1 thread/semana
- Reply a mínimo 10 cuentas de nicho/día
